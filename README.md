# Firman Hanafi — Portfolio

Personal portfolio website showcasing my professional profile, work experience, technical skills, and open-source projects.

## 🚀 Live

Deployed via GitHub Pages: _coming soon_

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, glassmorphism, responsive design
- **JavaScript** — Vanilla JS with Intersection Observer animations

## 📂 Structure

```
├── index.html        # Single-page portfolio
├── styles/main.css   # Design system & styles
├── scripts/main.js   # Interactions & animations
└── assets/           # Images & media
```

## 🏃 Local Development

Simply open `index.html` in your browser, or use a local server:

```bash
npx serve .
```
